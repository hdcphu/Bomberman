Project: Bomberman
Version: 0.1.0
Binary file: Bomberman.exe


How to play:
Control Player 1:
+ Direction: A S D W
+ Place bomb: B
+ Detonate a bomb: N

Control Player 2:
+ Direction: Keypad 4 5 6 8
+ Place bomb: Keypad 7
+ Detonate a bomb: Keypad 9

Pickup-items:
+ Red cube: Extent explosion
+ Black cube: More bomb
+ Yellow cube: Move faster in 10s
+ Blue cude: remote controlled bomb

*Character model is borrowed from a sample of Unity.
